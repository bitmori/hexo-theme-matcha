## Install

``` bash
hexo init Blog 
cd Blog 
npm install
npm install --save hexo-renderer-handlebars
git clone https://github.com/neonmori/hexo-theme-matcha.git themes/matcha
```

## Usage

Change `_config.yml` > `theme` -> `matcha`:

```yaml
theme: matcha
```

## Update

``` bash
cd themes/matcha 
git pull
```
